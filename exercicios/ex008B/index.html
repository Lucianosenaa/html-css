as tags para citações

Q, indica uma citação curta que não precisa de marcação de paragrafo.A maioria dos navegadores coloca o texto entre aspas.


BLOCKQUOTE=citação de bloco, indica uma citação longa. Normalmente, e renderizada visualmente por indentação, e usando essa tag o texto citando o determidado autor ficara com um espaço na lateral=deslocada horizontalmente.

e dentro de blockquote podemos criar um parametro usando a tag cite = blockquote cite="endereço de aum livro site etc", vai saber que tal frase estara citando tal livro ou site, link etc.


CITE, indica uma referencia a um trabalho artistico, como um titulo ou uma URL de referencia


para cita o autor de uma citação, pode-se usar a tag: footer, dentro da tag:CITE.
